MERN template

- all necessary boilerplate already done
- simple design layout for client part based on Ant design system

- Ant design system layout implemented

SERVER:

- Build on MVC architecture
- ES6 import enabled
- Express
- Mongoose
- Nodemon
- dotenv
- cors
